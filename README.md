# php-apache-composer

We are using this repository to maintain a Docker images on docker hub with `PHP 5.6.+` and supporting softwares to run and build `php symfony` application.

## Packages List 
* PHP 5.6.+
* Apache 
* PHP composer
* Git
* Zip
* python pip
* awscli
